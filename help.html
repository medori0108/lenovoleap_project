Help & Guide

Welcome to IT Events Hub! This page will help you understand how to use the website easily.

🔹 How to Use

1. Sign In: Enter your details to access the platform.

2. Dashboard: After login, you will see the main menu at the top right.

3. Post: If you are a company or organizer, you can post events.

4. View: Users can see all events and register.

5. Logout: Click logout to exit safely.

🔹 Tips

✔ Always fill correct event details while posting.

✔ Check event date and city before registering.

✔ Use the back button to navigate easily.

🔹 About This Website

IT Events Hub helps students, developers, and job seekers find hackathons, seminars, workshops, and job fairs in one place.
